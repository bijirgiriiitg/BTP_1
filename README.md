# BTP_1
## Photocatalytic Synthesis of Heterocyclic Compounds 

• Developed a visible-light photocatalytic method using Na2 eosin Y for the synthesis of benzo[c]chromeno[4,3,2-
gh]phenanthridine derivatives through hetero-cross-coupling cyclization and aromatization.
• Optimized key reaction parameters including solvent, photocatalyst, light source and reaction time to achieve an
86% yield.
• Used Column Chromatography and confirmed structures using 1H NMR, 13C NMR, IR spectroscopy and XRD.
